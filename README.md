This is another robotic arm project.

Hardware:
- SBC: Raspberry Pi 4B
- 4 MG90 servos
- Custom power PCB for servos (soon)

Software:
- movement_functions.py: defines servo movment logic and functions
- working_main.py: serves as executable file where movement function is called

Design:
- 3D printed frame designed in Fusion360 and printed on A1 Mini
